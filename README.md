Demo
=======
http://paveleremin.github.io/githubapi-with-angular/

Required tools
========
**nodejs** with **npm**  
**grunt** with **grunt-cli**  
**bower**

Install and launch the app
========
run this commands in terminal
```
npm install  
bower install  
cp app/_configuration/env.js-sample app/_configuration/env.js  
grunt serve
```
